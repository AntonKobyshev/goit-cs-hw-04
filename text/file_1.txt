В даній публікації розглянуто аналіз можливостей мови програмування Python для роботи
з просторовими даними. Автор публікації: М.А.Кухар
У роботі проводиться детальний аналіз використання сучасних програмних інструментів
для роботи з просторовими даними. Досліджуються можливості, які пропонує мова
програмування Python для оптимізованої обробки таких даних. Оцінюються переваги цієї мови
програмування та наявних засобів, таких як ArcGIS та ArcPy, у контексті роботи з геопросторовими
даними. Головною метою дослідження є збір інформації щодо можливостей обробки просторових
даних за допомогою мови програмування Python.
ESRI вибрала Python як один з основних інструментів розробки через його здатність легко
виконувати більшість завдань користувачів за допомогою написання відповідних скриптів.
Завдяки використанню бібліотеки ГІС-функцій ArcPy, доступної з випуску ArcGIS 10, значно
полегшено створення скриптів для звичайних ГІС-спеціалістів.
У ході дослідження були розглянуті загальні риси мови програмування Python. Особливості
цієї об'єктно-орієнтованої мови програмування були виявлені, і визначено способи її застосування
для роботи з геопросторовими даними, використовуючи пакет ArcPy, який є складовою
геоінформаційної системи ArcGIS.
Практичним результатом даного дослідження є презентація можливостей роботи з
просторовими даними в контексті інтерпретатора Python у таблицях атрибутів, а також наведення
прикладу використання функцій пакету ArcPy, таких як Describe, UpdateCursor та ListLayers, для
роботи з електронними картами.
Для себе у подальшій роботі вважаю корисною інформацією, що при використанні Python
в ArcMap можливо проводити роботу з шейпфайлами. Можливо виводити елементарну
інформацію таблиці атрибутів шейпфайлу, її також можливо змінювати, наприклад за допомогою
функції UpdateCursor. Також корисно що інтерпретатор Python в ArcMap дозволяє проводити
операції з самими електронними картами. Наприклад, змінювати їх прозорість.
Висновки:
У статті було проведено аналіз можливостей використання сучасних програмних засобів
для обробки просторових даних. Загальний аналіз мови програмування Python показав, що ця
мова є доречною для обробки геопросторових даних. Аналіз функцій, вбудованих у пакет ArcPy,
довів практичну цінність використання Python для обробки просторових даних, та підтвердив, що
ArcPy має необхідні інструменти для цього. Концептуальне представлення можливостей
